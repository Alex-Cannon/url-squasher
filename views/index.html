URL Squasher

Shorten long URLs with this Microservice!

For Users -

Simply use the form below to get a shortened URL. Simple and Awesome.

You can visit this link at any time to be redirected to your long url. Life's too short to type all those characters. Hahaha.

For Devs -

Shoot a GET request to the link below to get a shortened URL.

https://url-squasher.glitch.me/new/https://your-long-url.com/

Example Usage:

GET -> https://url-squasher.glitch.me/new/https://your-long-url.com/

JSON -> {old_url: "http://old-url.com", short_url: "https://url-squasher.glitch.me/url_id"}
